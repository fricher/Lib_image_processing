## image_processing :
Image processing library developped for the saliency map babbling

---

## How to Install :

# Dependancies:
- pcl-1.7.2 compile with C++11
- OpenCV 2 and Opencv nonfree

# Installation of image processing library :

follow the command lines :
<pre>
cd image_processing
mkdir build
cd build
cmake ..
make
sudo make install
</pre>
